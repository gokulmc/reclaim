Menu bar icon — pick a concept

All concepts are drawn as template icons (monochrome, auto-adapts to menu bar light/dark) and turn amber/red only when disk space runs low — same behavior as today, new face. The free-space number stays next to the icon in every concept except C-solo/D (noted below).

A · Return arrow → driveRECOMMENDED

An arrow hooks back down onto a drive slab: “space, returned.” Mini version of the app icon, so menu bar and Dock read as the same product.

B · Freshly-cleaned drive

Drive slab with a sparkle — “this disk was just tidied.” Friendliest metaphor; less unique (sparkles are everywhere in 2026 UIs).

C · Disk gauge ring

A ring fills as your disk fills — glanceable health meter, like a tiny battery for disk. Works icon-only (no number) or with the number.

D · Number only

No icon at all — just “54 GB”. Cleanest possible footprint, but loses identity and the color warning has nothing to tint except text.
